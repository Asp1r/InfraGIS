# ROAD GIS Quickstart

Короткая памятка для оператора/администратора InfraGIS (этап 1).

## 1) Что уже есть в системе

- `Дорога` (`kind=road`) — корневой слой.
- `Ось дороги` (`kind=road_axis`) — линейный референс.
- `IRI` (`kind=iri`) — слой продольной ровности.
- `Дефекты` (`kind=defects`) — слой дефектов.
- `360-кадры` — можно привязать к оси дороги с расчетом `axis_km`.

Все диагностические записи привязываются к оси через километраж (`km_start`, `km_end`).

## 2) Ежедневный рабочий поток

### Шаг 1. Создать дорогу

1. Открыть `Админ -> Слои`.
2. Заполнить название.
3. Нажать `Создать дорогу`.

### Шаг 2. Загрузить ось

1. Выбрать созданную дорогу.
2. Загрузить файл или URL оси.
3. Поддерживаемые форматы:
   - `.geojson`
   - `.csv`
   - `.zip` (Shapefile: `.shp + .shx + .dbf`)
4. После загрузки проверить:
   - `total_km > 0`
   - число точек `points` адекватно геометрии

### Шаг 3. Проверить дерево

У дороги должны быть доступны вложенные слои:

- `Ось дороги`
- `IRI`
- `Дефекты`

### Шаг 4. Добавлять диагностику

- IRI: записывать значения по участкам (`km_start`, `km_end`, `iri_value`).
- Дефекты: записывать тип, тяжесть, норматив (`norm_ref`) и диапазон км.

### Шаг 5. Привязка 360 к оси

1. В модуле 360 создать geolink точки (`lon`, `lat`).
2. Передать `axis_layer_id` (или `layer_id`, из которого выводится ось).
3. Убедиться, что у точки появился `axis_km`.
4. После обновления оси выполнить bulk-пересчет `axis_km` (admin endpoint).

## 3) Обязательные проверки перед сохранением

- `km_end >= km_start`
- `axis_layer_id` указывает на слой `road_axis`
- координаты оси корректны и в нужном порядке по трассе
- данные не выходят за длину оси (`total_km`)

## 4) Частые ошибки

- `axis_layer_id is required`
  - Добавьте `axis_layer_id` при создании IRI/дефекта.

- `Layer is not a road axis`
  - Передан неверный слой в `axis_layer_id`.

- `CSV must contain lon/lat columns`
  - В CSV нет координатных колонок (`lon/lat` или синонимов).

- `Shapefile ZIP must include .shp, .shx and .dbf`
  - Архив неполный.

- `axis_layer_id must point to road_axis`
  - В geolink передан слой не типа `road_axis`.

## 5) Полезные endpoint-ы

- `GET /layers/tree` — дерево дорожных слоев
- `POST /layers/roads` — создать дорогу
- `POST /layers/{road_id}/axis` — загрузить ось + получить `total_km`
- `POST /layers/iri-records` — создать запись IRI
- `POST /layers/defect-records` — создать запись дефекта
- `POST /media360/items/{media_id}/geolink` — привязать 360-точку и рассчитать `axis_km`
- `GET /media360/map-points?axis_layer_id=...` — получить 360-точки по оси
- `POST /media360/axis/{axis_layer_id}/recalculate-km` — массово пересчитать `axis_km` (admin)
